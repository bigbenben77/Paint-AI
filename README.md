# 画图AI版


2026年，当别人还在卷Midjourney、抢Stable Diffusion的额度时——
我，用PyQt5，手搓了一个画图。


不是GPT画的，不是Copilot写的，是我一行行码出来的。

不是Web的，不是App的，是桌面端的、像素级的、能闪退的真·画图。

【功能整活】

✅ 支持17种AI大模型（从豆包到DALL-E，从百度文心到商汤秒画）

✅ 支持16种画图工具（铅笔、喷枪、多边形、任意形状裁剪）

✅ 支持Ctrl+C/V（这很重要，因为有些AI画图不支持）

✅ 支持橡皮擦右键精确替换前景色（懂的人已经开始哭了）

❌ 不支持图层（问就是复古）

❌ 不支持云端同步（你的大作只存在你的C盘）
